# Usage

<pre>
USAGE
  $ emuto [FILTER]

OPTIONS
  -c, --color    Color output
  -h, --help     show CLI help
  -u, --ugly     Don't prettify output
  -v, --version  show CLI version
</pre>

# Setup

```
npm install -g emuto emuto-cli
```
